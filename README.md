# Youtube-LyricFlow Chrome拡張機能
![Lyrics](/images/Lyrics.png) 

Youtube-LyricFlow は、YouTubeでの音楽視聴体験を向上させるChrome拡張機能です。<br>美しくインタラクティブなUIで歌詞や字幕を表示できます。<br>
💡`Status: 開発中`🚧

## アプリの概要

- **リアルタイム同期**: 再生状態に合わせて字幕を正確にハイライト＆スクロール。
- **心地よい UI/UX**: YouTubeのデザインに干渉せず、高品質なアニメーションと直感的な操作を提供。
- **堅牢なシステム設計**: 動画遷移に強く、外部サーバーに依存しない安定した字幕データを取得。
- **柔軟なカスタマイズ**: フォントや文字サイズの変更、翻訳歌詞の同時表示など、視聴スタイルに合わせた多様な設定が可能。

## 使用方法 (Quick Start)
**※現在、開発環境であるMacOSのみ動作確認済**

1. リポジトリをダウンロード
2. パッケージをインストールしてビルド
```bash
npm install
npm run build
```
3. Chrome への読み込み
   1. Chrome で chrome://extensions/ を開く。
   2. 右上の「デベロッパー モード」をオン。
   3. 「パッケージ化されていない拡張機能を読み込む」をクリックし、dist フォルダを選択。

## 開発環境 (Local Development)

```bash
git clone <URL>
cd Youtube-LyricFlow
npm install
npm run dev
```
※ 開発中は保存するたびに自動リロード。上記と同様の手順でChromeに `dist` を読み込む。

## 工夫点

YouTubeの巨大なSPA上での堅牢な動作と、没入感を削がない緻密なUI/UX設計の両立を、徹底的に追求。

### 1. 視覚的な心地よさを追求したUI
1ピクセル単位でのレイアウト調整、高品質なUIアニメーションとマイクロインタラクションを提供。
- ドロップダウンや各メニューが「ふわっと」開く、自然な出入りアニメーション。
- ドロップダウン内で、マウスの動きに吸い付くように追従する滑らかな選択アニメーション。
- どんな小さな操作にも、確かな手応えを感じられるマイクロフィードバック。

### 2. 使用体験を徹底的に追求したUX設計
ユーザーがストレスなく視聴に没入できるよう、細部までUXの工夫を凝らしています（以下はその一例）。
- **直感的なドラッグ操作:** 歌詞やメニュー画面など、主要なUIはすべてインタラクティブで自由にドラッグ配置が可能。
- **シームレスな画面遷移体験:** YouTubeの通常画面と全画面でのUIの使い勝手・見え方の差を無くし、一貫した使いやすい体験を提供。
- **さらなる操作の快適性:** UI操作より素早く、快適に調整できるよう、キーボードショートカットをサポート。
- **イントロ・アウトロの視覚的配慮:** 歌詞のない部分(イントロ・間奏・アウトロ)を自動検出し、歌詞がずっと画面に表示される不格好さを解消。不要な場面では視界を邪魔しない表示に。
- **多様なニーズへの対応:** 翻訳歌詞の同時表示や、ふりがな表示機能も完備。
**etc...**

### 3. パフォーマンスの最適化
   静的なUIコンポーネントの構築にはPreactを活用しつつ、毎フレーム実行される歌詞の同期処理にはVanilla JSでの直接DOM操作とCSS変数更新に限定。これにより60fpsの滑らかな同期スクロールを実現。

### 4. 効率的で堅牢な字幕データ取得アプローチ
   不安定なDOMスクレイピングを避け、ブラウザの標準API（PerformanceObserver）で、YouTubeの字幕リクエストを検知し、再利用する設計とした。この方式により、YouTubeのUI変更の影響を受けにくく、不要なスクレイピング負荷を発生させない設計になっている。

### 5. 巨大SPAでの完全にクリーンな体験と自己修復UI
   YouTubeのページ遷移なしの動画切り替えを完全にフックし、前の動画のデータが残るのを防いで、シームレスな遷移を実現。さらに、YouTube側で動画プレイヤーのDOMが突如再構築されても、UIの切断を監視して自動で再マウントする機構を実装し、拡張機能のサイレントクラッシュを防いでいる。
